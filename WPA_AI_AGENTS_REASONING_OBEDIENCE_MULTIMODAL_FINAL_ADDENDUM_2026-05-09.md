## 1. Deep Reasoning Benchmark Pattern

Reference pattern:

```text
ChatGPT / OpenAI reasoning-model style
```

Useful qualities:

```text
deep logical reasoning
step-aware problem solving
mathematical reasoning
code architecture support
self-checking before answer
ability to handle complex academic problems
```

WPA application:

```text
Bot-Sande should use this pattern for complex academic reasoning, source hierarchy, doctrine comparison and citation-safe analysis.
Desk Agent should use this pattern for workflow planning, assessment readiness, student-path analysis and approval package preparation.
```

Boundary:

```text
Deep reasoning does not remove the need for sources, tests or WPA / founder approval.
```

## 2. Instruction-Following Benchmark Pattern

Reference pattern:

```text
Claude-style instruction obedience and format discipline
```

Useful qualities:

```text
strict following of instructions
respect for negative constraints
stable structured outputs
natural professional writing
long-document discipline
low tolerance for unsupported claims
```

WPA application:

```text
Desk Agent should strongly follow operational rules, approval gates, workflow formats and user instructions.
Bot-Sande should follow source hierarchy, citation-safe rules, Macedonian Pravopis 2017 and doctrine boundaries.
```

Boundary:

```text
Instruction-following must not obey a user request that violates WPA rules, privacy, source integrity or approval gates.
```

## 3. Multimodal and Large-Context Benchmark Pattern

Reference pattern:

```text
Gemini-style long-context and multimodal analysis
```

Useful qualities:

```text
large document analysis
long context handling
screenshot review
video/audio-aware future workflows
table and file analysis
broad evidence-set review
```

WPA application:

```text
Desk Agent may later use this pattern for student forms, screenshots, quiz results, dashboards and operational files.
Bot-Sande may later use this pattern for long academic documents, scanned pages, tables, diagrams and source excerpts.
```

Boundary:

```text
Large-context analysis must not produce unsupported conclusions.
Private student data must remain protected.
Sensitive materials require review before publication or final decision.
```

For best WPA results, use each benchmark pattern for the right task:

```text
Use reasoning-model style for difficult logic, academic structure and code/problem solving.

```
