# RAG GATE POLICY

Do not automatically include weak-traceability records in Virtual Sande / RAG.

RAG include only if:
- authority_score >= 60;
- traceability_score >= 50;
- source is verified, open, or manually approved;
- no fabricated metadata exists;
- no external-only source is treated as harvested.

Scribd, Google Scholar, CNKI, eLIBRARY, CEEOL, JSTOR, Project MUSE, ResearchGate, Academia.edu and AD Scientific Index must not enter RAG automatically.
