# WPA Audio Video Creator Engine v9

Final functional build.

## Што содржи

- 20 активни agent stages
- 20 активни agent panels
- 200 WPA сценарија
- Scenario Bank 200 со search + category filter
- Right Command Panel со работни Quick Actions
- Live Rooms со hand queue, give floor, mute-all
- Delivery Center со bundle export, JSON export, QA report
- SOP Manual

## Како да го користиш

1. Отвори `wpa-audio-video-creator-engine-v9-final-functional.html`.
2. На Dashboard внеси project/topic/audience.
3. Оди во `Scenario Bank 200`.
4. Избери сценарио.
5. Кликни `Build Selected`.
6. Оди во `20-Step Workflow` и пушти Run Current или Run All.
7. Оди во `Delivery & Export` и симни bundle.
