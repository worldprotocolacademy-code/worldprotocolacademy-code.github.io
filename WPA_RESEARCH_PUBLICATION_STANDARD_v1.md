# WPA Research Publication Standard v1

World Protocol Academy — Institute for Protocol, Diplomacy, Public Communication & Security Studies

## Purpose

This document defines the minimum publication standard for WPA research outputs.

## WPA Research Output Types

1. WPA Working Papers
2. WPA Protocol Studies Review
3. WPA Policy Briefs
4. WPA Case Studies
5. WPA Annual Reports
6. WPA Global Protocol & Diplomacy Benchmark Methodology Papers

## Minimum Metadata

Every public WPA research item should include:

- Title
- Author
- Date
- Abstract
- Keywords
- WPA publication series
- Stable URL
- Citation format
- References
- PDF version when possible
- Canonical link
- Meta description
- Language code

## Recommended HTML head

```html
<meta name="description" content="...">
<link rel="canonical" href="https://worldprotocolacademy-code.github.io/...">
<meta name="robots" content="index, follow">
```

## Recommended Citation Format

Smiljanov, S. (2026). Title of paper. World Protocol Academy — Institute for Protocol, Diplomacy, Public Communication & Security Studies. URL.

## Editorial Rule

No WPA research page should make university-equivalence claims unless a formal university partnership or accreditation exists.
