Зошто WPA Identifier System

До официјални ISSN/ISBN/DOI, WPA користи транспарентен внатрешен систем за архивирање, цитирање и governance.

Природа на WPA IDs

WPA Identifier е внатрешен институционален идентификатор. Тој не е ISBN, ISSN или DOI.

Целосна табела на ID формати

ФорматТипПример
WPA-J-YYYY-NNJournal issueWPA-J-2026-01
WPA-J-ART-YYYY-NN-XXXJournal articleWPA-J-ART-2026-01-001
WPA-WP-YYYY-XXXWorking paperWPA-WP-2026-001
WPA-PB-YYYY-XXXPolicy briefWPA-PB-2026-001
WPA-AI-DISC-YYYY-XXXAI Transparency noteWPA-AI-DISC-2026-001
WPA-CORR-YYYY-XXXCorrection noticeWPA-CORR-2026-001
WPA-RET-YYYY-XXXRetraction noticeWPA-RET-2026-001

Објаснување на ID полиња

WPA = World Protocol Academy; J = Journal; ART = Article; YYYY = year; NN = issue sequence; XXX = item sequence.

Примери

WPA-J-ART-2026-01-005 = петта статија во прв број, September 2026.

Sample citation со WPA ID

Pre-DOI citation
Author, A. A. (2026). Title. WPA Journal, 1(1). WPA Identifier: WPA-J-ART-2026-01-001. DOI not yet assigned.

Retroactive DOI assignment

Ако WPA добие DOI infrastructure, DOI се додава retroactively како дополнителен идентификатор; WPA ID останува.

ISSN секција

ISSN се бара по прв број и eligibility. Никакво misleading ISSN pending.

ISBN секција

ISBN важи за books/monographs/proceedings, не за journal articles.

DOI секција

DOI will be assigned only after formal setup through Crossref or another recognised DOI registration agency.

ID assignment workflow

  1. Submission ID internal.
  2. Accepted article gets WPA Article ID.
  3. AI note/correction IDs are created where needed.
  4. WPA ID remains immutable.

ID validation rules

Uppercase ASCII, hyphen-separated, zero-padded sequences, unique and never reused.
