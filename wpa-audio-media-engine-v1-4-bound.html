      Guide

      How to use the engine efficiently.

      1. Select or Create Project

      Everything attaches to the active project: history, notes, outputs and delivery.

      2. Work in Modules

      Audiobooks, Voice, Content, Viral, Protocol and Production are the creative cores. Delivery and Monetization close the loop.

      3. Create Output Cards

      Each major module can push polished snippets to the dashboard and delivery center.

      4. Save, Import, Export

      The engine uses local storage. You can export the full state or import a prior JSON state.

      Shortcuts

      Ctrl/Cmd+S = Save. Search bar scans projects, books, chapters and scenarios. Import restores prior engine state. Guide and Demo are always available from the left menu.
