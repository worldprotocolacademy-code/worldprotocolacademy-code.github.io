## Core principles

1. The Board is advisory only.
2. The Board does not create accreditation, university status, recognition, or licensing status.
3. The Director remains executive authority.
4. Board members provide independent academic, professional, ethical, and methodological advice.
5. Board review is not the same as external peer review.
6. The Board should strengthen realism, quality, privacy, certification integrity, and anti-overclaiming discipline.
