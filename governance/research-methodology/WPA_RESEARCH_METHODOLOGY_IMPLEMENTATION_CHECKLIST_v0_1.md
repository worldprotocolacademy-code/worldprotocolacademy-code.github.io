# WPA Research Methodology v0.1 — Implementation Checklist

**World Protocol Academy**  
**Institute for Protocol, Diplomacy, Public Communication & Security Studies**  
**Classification:** WPA-INTERNAL

## Phase 1 — Adopt internal draft

- [ ] Director reviews WPA-METH-001 v0.1.
- [ ] Director confirms it is an internal methodology draft, not public marketing.
- [ ] Store final internal draft in `governance/research-methodology/`.
- [ ] Do not add to sitemap.
- [ ] Do not link from public navigation.

## Phase 2 — Apply to new outputs

For every new WPA research output:

- [ ] Define the research question.
- [ ] Identify publication category: working paper, policy brief, case study, protocol analysis, benchmark methodology, monograph, proceedings.
- [ ] Identify source types.
- [ ] Verify citations.
- [ ] Check that no citation is invented or AI-generated without verification.
- [ ] Add AI-use disclosure if AI tools were used.
- [ ] Add conflict-of-interest statement where relevant.
- [ ] Add methodology or limitations note.
- [ ] Complete internal review.
- [ ] Decide whether external review is needed.

## Phase 3 — Review existing outputs

- [ ] Identify WPA outputs that are active/public and likely to be cited as authority.
- [ ] Prioritize books, papers, and benchmark-related pages.
- [ ] Check for overclaiming.
- [ ] Check citation quality.
- [ ] Check whether external sources are sufficiently engaged.
- [ ] Add corrections/changelogs where needed.
- [ ] Avoid claiming retroactive compliance.

## Phase 4 — Advisory Board review

- [ ] Send executive summary to Academic Advisory Board candidates.
- [ ] Request comments on methodology.
- [ ] Ask for review of source standards, AI-use disclosure, case study method, peer review language, and certification claims.
- [ ] Record recommendations.
- [ ] Prepare Version 0.2 or Version 1.0 after review.

## Phase 5 — Publication discipline

Before publishing any WPA research page or paper:

- [ ] Check title.
- [ ] Check author.
- [ ] Check date.
- [ ] Check abstract.
- [ ] Check keywords.
- [ ] Check citations.
- [ ] Check bibliography.
- [ ] Check source verification.
- [ ] Check AI-use disclosure.
- [ ] Check conflict-of-interest statement.
- [ ] Check limitations.
- [ ] Check language quality.
- [ ] Check that no false accreditation/recognition claim exists.

## Do not do

- [ ] Do not call internal review “peer review.”
- [ ] Do not claim accreditation unless formally granted.
- [ ] Do not cite AI as a source.
- [ ] Do not publish unverified citations.
- [ ] Do not represent aspirations as achieved status.
- [ ] Do not treat founder-authored sources as automatically superior to external scholarship.
