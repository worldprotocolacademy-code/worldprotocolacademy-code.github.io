About this package

This page provides access to the consolidated WPA Global Institutions Master List v1.0-CORRECTED-4E. The package includes the master dataset, public-readable Markdown files, summary PDF/DOCX documents, analysis files, QA report and changelog.

The dataset is intended for internal review, research structuring, institutional mapping and further source verification. It is not presented as a final public ranking, accreditation list or verified official benchmark.

Status: Pre-publication candidate / internal review. Not final public benchmark until source and visual verification are completed.
Key correction: A010 is treated as a reported affiliated branch / historical branded training presence, Level B, Verification pending. A005 and B008 are treated as reported cooperation-model observations, not verified institutional entities.
