# WPA Admin Security Checklist

## 15 Rules with Consequences

| # | Rule | Consequence |
|---|---|---|
| 1 | 2FA mandatory | Suspension until activated |
| 2 | Password change every 90 days | Written warning |
| 3 | No public WiFi for admin access | Warning / suspension |
| 4 | Password manager required | Warning |
| 5 | No account sharing | 30-day suspension |
| 6 | No personal email for certificates | Suspension |
| 7 | No public AI tools for user data | Permanent revocation for serious cases |
| 8 | Report suspicious login immediately | Warning |
| 9 | WPA-approved devices only | Suspension |
| 10 | No downloading user data without reason | Suspension |
| 11 | All grade changes documented | Suspension |
| 12 | All QR/certificate changes logged | Suspension |
| 13 | Lock session when away | Warning |
| 14 | WPA Security Training before access | No access without training |
| 15 | All admin activities audit-logged | No exceptions |

**Consequences escalation:**
- 1st violation = Written warning
- 2nd violation = 30-day suspension
- 3rd violation = Permanent admin privilege revocation
