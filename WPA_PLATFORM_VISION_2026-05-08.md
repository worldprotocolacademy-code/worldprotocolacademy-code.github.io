## Purpose

This document records the high-level platform vision for World Protocol Academy and WPA Institute.

No production code is modified by this document.

## Vision

WPA should grow into a professional AI-supported platform for protocol, diplomacy, public communication, security studies and institutional education.

## Main Layers

```text
Bot-Sande = academic knowledge brain.
Desk Agent = operational workflow brain.
WPA Knowledge Vault = source foundation.
Workflow Agents = future specialized support layer.
WPA / founder approval = final institutional control.
```

## Bot-Sande

Bot-Sande focuses on academic knowledge, source-based answers, definitions, comparisons and doctrine protection.

Primary sources include books, doctoral dissertation, papers, interviews, WPA materials and verified secondary sources when needed.

## Desk Agent

Desk Agent focuses on orientation, student pathway, operational guidance, question-bank practice, review preparation and handoff to Bot-Sande when academic knowledge is needed.

## Knowledge Vault

The WPA Knowledge Vault should organize books, papers, interviews, official materials, question-bank materials, terminology, language rules and translation memory.

## Review-Ready Outputs

The platform should produce clear review-ready outputs with:

```text
request summary
known information
missing information
source or rule basis
recommended next step
approval needed: yes/no
handoff needed: yes/no
```

## Final Principle

```text
Bot-Sande is the academic brain.
Desk Agent is the operational brain.
WPA Knowledge Vault is the source foundation.
WPA / founder approval remains the final institutional control.
```
